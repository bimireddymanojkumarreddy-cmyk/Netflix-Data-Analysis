# 🎬 Netflix Data Analysis using Python

## 📌 Project Overview

This project performs an end-to-end Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset using Python. The objective is to uncover meaningful business insights, identify content trends, and understand Netflix's content distribution across different countries, genres, ratings, and years.

The project demonstrates practical data analysis skills including data cleaning, feature engineering, visualization, and business insight generation.

---

## 🚀 Project Objectives

* Clean and preprocess the Netflix dataset
* Perform Univariate Analysis
* Perform Bivariate Analysis
* Explore content distribution across countries
* Analyze Movies vs TV Shows
* Study release trends over the years
* Analyze content ratings
* Discover the most common genres
* Generate business insights from visualizations

---

## 📂 Dataset

## 📈 Key Business Insights

* Movies dominate Netflix's content library.
* The United States contributes the highest amount of content.
* Drama and International Movies are among the most popular genres.
* Significant growth in Netflix content production occurred after 2015.
* TV Shows generally have fewer releases than Movies but have grown steadily over recent years.
* Most Netflix content is targeted toward mature audiences (TV-MA).
